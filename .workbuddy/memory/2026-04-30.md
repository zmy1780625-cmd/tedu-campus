- **部署方式**：用户提供 PAT（ghp_xxx）→ curl REST API 创建仓库 + 启用Pages → git push（HTTPS+token认证）→ token从remote URL清除
- **关键命令**：
  - 创建仓库：`POST /user/repos`（描述用英文避开中文JSON转义问题）
  - 推送：`git remote add origin https://TOKEN@github.com/USER/REPO.git`
  - 启用Pages：`POST /repos/{owner}/{repo}/pages` body=`{"source":{"branch":"main","path":"/"}}`
  - 安全收尾：`git remote set-url origin https://github.com/...`（移除URL中的token）
